Glossary
What should you expect?

In this section, we will be looking at 20 explanations of different web development and computing terms. This will hopefully build a familiarity with some of the computing terms. For more information please visit: https://www.wholewhale.com/tips/developer-terms-glossary

# Known as error message (404) that pops up when the link isn’t working because it is broken.

Application Known more commonly as an app, are created to perform tasks.

Attribute A characteristic of your website design. For example, color.

Back End Keeps the front end of the website running, it’s what users of websites don’t see, the coding as far as the coding aspect.

Browser Programs like Google Chrome that allow the user access to the web.

Cache Data that is already stored from previous searches that allow for quicker retrievals. Will help with faster load speed from repeat website visitors.

Classes Acts an identifier in CSS of what is exactly is being targeted when styling your front end. For example, titlewhich will only target content inside of this class.

Domain Defined by their IP addresses, domains are the address of a website, for example www.google.com

Firewall A proxy would be one example, is a form of protection that will prevent an insecure network enter a secure network.

Front-End Opposite to back-end, is what the user does see, with the different fonts and images.

GUI Initialised for Graphical User Interface, is image which the user can interact with for example on your desktop their icons that you can click which will start up a program.

HTML Initialised for Mark-up Language, is the language used in coding in web development (building a website).

Meta Tag Additional data on websites, this piece of content will appear in the google search, for example the main keywords within the search that appear in the website.

Navigation Allows the user to move from one page to another in the same website, for instance a developer would be able to link his pages through hyperlinks.

Opening Tag (associated with closing tags) The tags (<>) help build the structure of the site.

Property In terms of CSS, the property that are determined by CSS for example colour schemes and fonts.

Resolution Measured in pixels, show the size of the image.

Server Takes the requests of users and delivers them from the internet to them, acts as the gap between the user and the web.

UI The User Interface, are the visuals that are in a website.

Wireframe The simplest, basic design of a website without colours or fonts, just displays the simple foundation.
