Purpose of the Site

This newspaper-based website was created as part of my assignment for web development for my first term at university, to showcase everything I have learned, that I both struggled and achieved as part of my module in computing. For the user, this site was created to be responsive so that not only can it be used on desktop devices but also mobile devices.

This assignment, despite its difficulty from my lack of skill and knowledge, was my favourite one I have received from all my assignments, I have spent most of my time reading what I can about it since I have always been curious about how websites, games and other web resources are created.

What is Web development?

Web development is the creation of programs or applications that are used in the internet. A web developer would therefore need to be able to create these resources. We will go deeper on this in the Jobs section looking at the skills needed specifically for this position.
