What should you expect?

Bootstrap

An open source toolkit which is used for development of HTML, CSS and JS.

This review was given by Validated Reviewer named Robert S., here on this page will provide his opinion of his experience with using Bootstrap as a web-kit tool.

Robert S Review

"A good framework for responsive, modern and fast design of websites"

What do you like best?

"It is a good framework if you want to develop websites that are responsive quickly and effectively. To be able to apply it only to use the boostrap's own classes and the web is stamped in a nonsense. Bootstrap does not require any type of installation on the server or the computer, just include the bootstrap libraries in your project and it can already be used. Being a very extensive framework, there is a large support community that can help you if you have a problem with the implementation of its components".

What do you dislike?

"The bootstrap components are pretty basic, if you want to make a website with very special features and very specific features you have to look for life to do it, as it includes it. The bookstores should be placed within the code in the manner indicated in the bootstrap documentation but may have problems to work correctly. Websites may take a while to load."

Recommendations to others considering the product

"I highly recommend all the professionals who need to make the front-end of the webs quickly, responsive and with a very visual and pleasant design".

What business problems are you solving with the product? What benefits have you realized?

"I use it to design web pages responsive because it allows me to develop websites very quickly and that are very good. Customers ask for this type of framework because it is very visual and adapts to all types of devices".

(Robert, S. 2013) source for the review: https://www.wholewhale.com/tips/developer-terms-glossary
